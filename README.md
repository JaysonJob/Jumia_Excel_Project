# Jumia_Excel_Project
This is a personal data analysis project where i used microsift excel to explore and analyze data from Jumia.It's one of my early projects i built in my data analysis skills

## What this project is about
I wanted to practise working with real-world data so i created a jumia dataset and used excel to clean it up,analyze and generate some meaningful insights.The goal was to answer basic structure questions like which product had the highest rank.

## What I Did

- Cleaned up the raw data (removed duplicates,filled in blanks)
- Used Excel formulas like `SUM`, `AVERAGE`,and `IF` to do calculations
- Built pivot tables to summarize the data in different ways
- Created charts to visualize trends and make the numbers easier to understand
- Used the data to answer basic business questions about product rankings and sales

## What I Learned

This project taught me how messy real data can be and how much work goes into cleaning it before you can start analyzing. I also got more comfortable with pivot tables and Excel formulas along the way.
